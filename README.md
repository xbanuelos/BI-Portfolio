# BI Porfolio - Ximena Bañuelos
Data analytics projects focused on e-commerce, logistics, fintech,
and healthcare domains - inspired by the analytical challenges of
companies like Rappi, DiDi, Uber, Mercadolibre, and Nu.

## What´s in here
50 projects covering the full BI stack: exploratory data analysis, SQL,
dashboarding in Power BI, and interactive apps built with Streamlit. 
Each projects starts from a real business question and ends with an actionable insight.

## Tools & stack
Python (pandas, matplotlib, plotly, sckit-learn)
SQL
Powe BI
Streamlit
Git

## Background
I came from Biomedical Engineering, wich means I think about data
with a research mindset - careful with assumptions, rigorous with
cleaning, and always asking why before how. Most projects have a 
health domain varian alongside the e-commerce version.
